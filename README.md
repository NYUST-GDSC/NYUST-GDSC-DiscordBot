# NYUST GDSC Dart Discord Bot

---

## 核心技術 Core Technic

> Dart Nyxx Discord Bot

---

## 快速上手 Getting Start


### Step.1 安裝套件

> dart pub get 

### Step.2 加入 Discord Channel Token

建立 .env

```shell
BOT_TOKEN = "XXXXXXXXXXXXX"
```

### Step.3 執行程式

```
dart run
```
